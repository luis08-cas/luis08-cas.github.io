## Hardware


El hardware es toda la parte física de un computador, es decir todo lo que se puede ver y tocar, y esta dividido por varios componentes, tales como:

- **Internos**
    - Unidad Central de Procesamiento (CPU)
    - Memoria RAM
    - Disco duro o unidad de estado sólido (SSD)
    - Placa base (placa de circuito impreso)
    - Tarjeta grafica 
    
- **Entrada**
    - Teclado
    - Mouse
    - Micrófono
    
- **Salida**
    - Monitor
    - Impresora
    - Altavoces
    
## Software

`El Software es toda la parte lógica de un computador, y la que trabaja de la mano con el **Hardware** para que la computadora funcione correctamente, sin la parte lógica el Hardware solo seria una pieza de metal y plástico inservible

- **Componentes y funciones principales**

   - ***Instrucciones:*** Conjunto de comandos que guían al hardware para ejecutar funciones concretas.

   - ***Programas:***  Conjunto de instrucciones para realizar tareas específicas, como un procesador de texto o un juego. 

   - ***Aplicaciones:***  Programas diseñados para cumplir un propósito específico del usuario, como WhatsApp, Netflix o un software de diseño.

- **Tipos comunes de software** 

    - ***Software de sistema:***   Se encarga de gestionar el hardware y los recursos del dispositivo, como los sistemas operativos 
    
    - ***Software de aplicación:*** Programas diseñados para que el usuario realice tareas concretas, como procesadores de texto, hojas de cálculo, navegadores web y aplicaciones móviles. 
    
    - ***Software de programación:***  Herramientas que los desarrolladores utilizan para crear otro software, como editores de código y compiladores. 

# Programa

# Variables

 Una variable es un nombre que usamos para hacer referencia a un valor que guardamos en la memoria del computador, existen varios tipos de variables, tales como:

- ***Enteros (int)***: Son las que guardan los números sin los decimales.

-  ***Flotantes(float)***: Son los que guardan los números con decimales.

-  ***Cadenas de texto (str)***: Guardan textos o palabras.

-  ***Booleanos (bool)***: Es el que guarda los valores de verdad, es decir verdadero o falso.

-  ***Listas (list)***: Es el que guarda valores dentro de una misma variable.


En resumidas palabras, las variables son la base de todos los programas, porque nos permite almacenar diversa información para poder manipularla y volverla a usar en el momento en que se necesite, debemos recordar que sin el uso de estás variables el programa nunca podrá recordar ni realizar cálculos.
